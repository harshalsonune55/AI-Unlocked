# Voyager 🌍✨

Voyager is an AI-powered travel assistant built with **React**, **Vite**, and **Tailwind CSS**.  
It helps users generate trip plans, itineraries, and smart travel ideas with a modern AI-style interface.

---

## 🚀 Features

- 🎨 Premium gradient AI-style landing page
- ⌨️ Typing animation in input placeholder
- 🧊 Glassmorphism input design
- 📱 Fully responsive layout
- ⚡ Fast development with Vite

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React (icons)

---

## 📦 Installation

Clone the repository:

```bash
git clone <your-repository-url>
cd voyger
````

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Project

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173/
```

---

## 🌈 Tailwind Setup (If Needed)

Make sure `src/index.css` contains:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Ensure `main.jsx` includes:

```js
import "./index.css";
```

---

## 🧹 Fix Permission Errors (Mac Only)

If you get `EACCES` or permission errors:

```bash
sudo chown -R $(whoami):staff .
rm -rf node_modules
npm install
```

Then run:

```bash
npm run dev
```
